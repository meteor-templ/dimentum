# Dimentum

Directed momentum animation.

## Install

```
meteor add templ:dimentum
```

## Documentation

### Example

```html
{{#momentum plugin='dimentum'}}
    {{#each documents}}
        <div data-dimentum='right'>
        </div>
    {{/each}}
{{/momentum}}
```
